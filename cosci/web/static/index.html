    Readout

    An instrument for hypothesis evolution.

    Define a research goal and run the engine. It generates hypotheses, grounds and reviews them, ranks them through an Elo tournament of simulated scientific debates, evolves the strongest, and synthesizes an overview. Select a run from the archive, or start a new one.

    Quality is a measured quantity here: the Elo trajectory rises as the system spends more compute — the paper's central result, plotted live.
